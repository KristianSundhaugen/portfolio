Developing an video-shearing website

My first encounter with Front-End Web Development

Summary

  • The Project Assignment

    • The assignment was to develop a website, where users could upload videos, make playlists and add videos to them. With the posibility for users to dynamically make subtitles for videos. The application needed also to have a login/register user function, where information would be stored securily in a database. Also with the posibility to have adminusers added, that has the power to delete any video or ban users. The biggest task for us was to make the whole application single page. This was done through ajax and jQuery.
