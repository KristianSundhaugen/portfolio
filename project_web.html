WWW-Technology Project

An NTNU Gjøvik School Project

Developing an video-sharing website

My first encounter with Front-End Web Development

Summary

  • The Project Assignment

    • The assignment was to develop a website, where users could upload videos, make playlist's and add videos to them. With the possibility for users to dynamically make subtitles for videos. The application needed also to have a login/register user function, where information would be stored securely in a database. Also with the possibility to have admin-users added, that has the power to delete any video or ban users. The biggest task for us was to make the whole application single page. This was done through jquery ajax.

  • Our work

    • One of our main focuses for this application was to build an single page application, with focus on object oriented code. Our application was built on the idea to have a main page where all the content appeared. This was done through the use of jquery ajax, it was a requirement that we used jquery ajax for this project, otherwise we were looking at react and angular as possibilities.

    We have a db_class that takes all requests towards the database, this can be for example a video, username, subtitles for a video or passwords. Passwords was of course handled properly through hashing. So the cooperation that makes the dataflow of content work, is that we have an html index file, where all the content ends up dynamically through a hashvalue change. Every html file, exept from index has a URL with an unique hashvalue. This is so we can call the content we want. Every html file has a cooperation with an script and an php file, that has their specific needed functions, used depending on what they need. This is then connected to database, where all of the variables stored from script and php, then receives the values fitting to the request needed towards the database.

    Videos where to be shown listed down the page. With the possibility to add them to your own playlist's. Other than that there was a function to add subtitles to your own videos. This was done through making an new text file, and dynamically adding components to this file, starting with a WebVTT. Then setting the start point for the caption where the user paused the video, letting them type a text in an field, then adding a 2 second's to the stop of the subtitle. This would then be sent to the database, where the id of the video was connected to the specific subtitle file. The user could also give the subtitles different languages, making the same video have multiple choices of subtitles. This was mainly done through javascripting.

  • What I have learned

    • I got an great introduction to jquery ajax, even though I would rather use something like react or angular, getting an basic knowledge of how jquery works is great. I also got good practice in javascriping, css and html and object-oriented code development. This project was what really caught my eye for Front-End development. I finally found a passion for an development style, but unfortunately got a very short time working with it in school. Making my knowledge basic, but the eagerness to learn more is great.
